# 题目

> medium 数学

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

**示例 1:**

> 输入: 121
> 输出: true

**示例 2:**

> 输入: -121
> 输出: false
>
> 解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

**示例 3:**

> 输入: 10
> 输出: false
> 解释: 从右向左读, 为 01 。因此它不是一个回文数。

**进阶:**

你能不将整数转为字符串来解决这个问题吗？

# 题解

通过 % 与 / 来比较数字两端的数字，比如 `199`，`a = 199 % 100 = 1` 与 `b = 199 / 10 = 9` 比较

```java
class Solution {
    public boolean isPalindrome(int x) {
        if (x < 0) {
            return false;
        }
        
        int tem = x; //首位数字 300 -> 3
        int help = 1;// x 的位数，300 -> 100
        while (tem >= 10) {
            tem /= 10;
            help *= 10;
        }
        
        while (x != 0) {
            if (x % 10 != x / help) {
                return false;
            }
            x = x % help / 10; //去掉最高位和最低位数字 131 -> 3
            help /= 100;
        }
        
        return true;
    }
}
```
